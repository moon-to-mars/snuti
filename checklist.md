# 프로토타입 체크리스트

PTBM 기반 유아 ADHD 행동 교정 웹앱 — **UI 프로토타입(화면만)** 진행 체크리스트.
외부 연결(API·DB·인증·AI) 없이 목업 데이터로만 구성한다.

- [x] TypeScript `strict: true` 설정 확인
- [x] ESLint + 포매터 설정
- [x] 스타일 도구 설정 (예: Tailwind)

- [x] 디렉터리 골격 생성 (`pages` `components` `features` `data` `types` `hooks` `utils`)
